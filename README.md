# CMake Simple Builder

Расширение VS Code для автоматической сборки и тестирования CMake проектов.

## Автор
Евстигнеев Сергей Евгеньевич - M3106

## Возможности
- Однокомандная сборка и тестирование CMake проектов
- Последовательное выполнение: `cmake -B build`, `cmake --build build`, `cd build`, `ctest -V`
- Вывод процесса в реальном времени в отдельной панели
- Обработка ошибок с описательными сообщениями

## Использование
1. Открыть папку с файлами плагина в VS Code
2. Нажать F5
3. В появившемся окне откройте папку со своим проектом в VS Code
4. Откройте Command Palette (`Ctrl+Shift+P`) и введите "CMake: Build and Test Simple"


## Требования
- Установленный CMake и доступный в PATH
- Доступный CTest для тестирования
- VS Code версии 1.75.0 или новее

## Установка
1. Склонируйте этот репозиторий
2. Создайте папку в расширениях VS Code (обычно "C:\Users\ВашеИмя\.vscode\extensions")
3. Скопируйте туда файлы расширения
4. Перезагрузите VS Code
